## Student Course Enrollment Dashboard — Power BI

An interactive Power BI dashboard built to analyze student course enrollment data, demographic patterns, fee distribution, and enrollment trends. Designed to help educators and administrators make data-driven decisions at a glance.

---

## Dashboard Overview

This dashboard provides a comprehensive view of student enrollment data across two key datasets:

- **Personal Details** — Student demographics including name, age, city, and age group
- **Student Course Details** — Course enrollment info including course name, date of joining (DOJ), and fees

---

## Key Visuals & Insights

| Visual | Type | What It Shows |
|--------|------|---------------|
| Enrollment by Month/Year | Clustered Column Chart | Tracks how student enrollments trend over time |
| Course Distribution | Donut Chart | Breakdown of students across different courses |
| Students by City | Clustered Bar Chart | Geographic distribution of enrolled students |
| Fee Overview | Gauge | Visual indicator of total fees vs target |
| KPI Cards | Card Visual | Quick stats — total students, total fees, etc. |
| Filters | Slicer | Dynamic filtering by course, city, age group |

---

## Tools & Technologies

- **Power BI Desktop** — Dashboard design and data modelling
- **DAX (Data Analysis Expressions)** — Calculated measures and KPIs
- **Power Query** — Data cleaning and transformation
- **Microsoft Excel / CSV** — Source data

---

 **To view the dashboard:** Download `student_project.pbix` and open it with [Power BI Desktop](https://powerbi.microsoft.com/desktop/) (free).

---

## How to Run

1. Download and install [Power BI Desktop](https://powerbi.microsoft.com/desktop/) (free)
2. Clone or download this repository
3. Open `student_project.pbix` in Power BI Desktop
4. Explore the dashboard using slicers and filters

---

## Key Learnings

- Building interactive dashboards with multiple visual types in Power BI
- Writing DAX expressions for calculated columns and measures
- Connecting and transforming data using Power Query
- Designing user-friendly layouts with slicers for dynamic filtering
- Analyzing enrollment trends using date hierarchies (Year → Quarter → Month)
